Pour info :
  • Uploadez votre fichier message_frais.txt sans l'avoir modifié
  • Vous devrez modifier manuellement la colonne pour les agréments usines
  • Toutes les dates sont changées au format jj/mm/aaaa
  • Les lignes du message frais ayant les mêmes caractéristiques sont fusionnées, les qtés et poids sont additionnés
  • Une colonne "Date production" est créée et calculée par le programme :
    • Elle ne fonctionne que sur les lots de type "SA3011D"
    • Elle se met automatiquement à la bonne année de production
    • Vous devrez modifier les autres dates manuellement (SAP transaction MMBE)
  • Votre numéro de commande s'affiche dans le fichier Excel
  • Vérifiez que les quantités et le poids total sont cohérents avec votre commande
  • Ctrl+P renvoie la page prête à imprimer au format paysage et adapté à une page
  • Ce site est hébergé gratuitement sur pythonanywhere et doit être activé tous les 3 mois
  • Existe en version payant (abonnement pythonanywhere) pour $5 par mois sans besoin d'activation manuelle
